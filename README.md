# LangChain Fundamentals

This project contains beginner-friendly LangChain notebooks and examples for learning AI application development with LangChain.

## What is included
- Introduction to LangChain concepts
- Model integration examples
- Python-based examples for experimenting with LLM workflows

## Project structure
- `main.py` - main Python entry point
- `updatedlangchain/1-langchainintro.ipynb` - introductory notebook
- `updatedlangchain/2-modelintegration.ipynb` - model integration notebook

## Next steps
This repository is being updated step by step. More content and examples will be added as the learning continues.
